## 2. Ethereum ERC-8004 - Trustless Agents Standard

### Overview
ERC-8004 is going live on Ethereum mainnet soon (announced Jan 28, 2026). This is a foundational standard for cross-organizational AI agent economies.

### Three Core Registries

1. **Identity Registry**
   - ERC-721 based on-chain agent identifiers
   - Resolves to registration files describing agent capabilities
   - Portable, censorship-resistant, transferable

2. **Reputation Registry**
   - Client feedback signals (human or machine)
   - Raw data on-chain, scoring algorithms off-chain
   - Enables agent scoring services, auditor networks, insurance pools

3. **Validation Registry**
   - Independent verification hooks
   - Supports: staking, zkML proofs, TEE oracles, re-execution
   - Results stored on-chain for transparency

### Why It Matters
- Eliminates API key silos between companies
- Enables agents to discover and trust each other without pre-existing relationships
- Works on any L2 or mainnet
- Complements A2A (Agent2Agent) and MCP (Model Context Protocol)

### Implications for AgentMemory Protocol
- Cross-chain agent identity integration
- Persistent reputation across sessions
- Validation for memory attestations
- Foundation for trustless agent collaboration
