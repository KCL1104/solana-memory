### 【Shot 01】重複對話截圖 - 快速剪輯
**時間碼**: 00:00 - 00:15  
**類型**: 蒙太奇 / 動畫

**畫面描述**:
- 多個聊天介面截圖快速切換
- 每次截圖顯示 AI 問相同的問題
- 數字計數器從 1 跳到 47

**技術需求**:
- [ ] 準備 5-7 個不同的對話截圖
- [ ] 使用 AE / FCP 製作數字跳動動畫
- [ ] 添加 Glitch 故障效果在第 47 次

**素材準備**:
```
截圖內容：
1. "你喜歡喝咖啡嗎？"
2. "請問你的咖啡偏好是？"
3. "你通常喝什麼咖啡？"
4. [重複類似問題...]
```

### 【Shot 04】聊天介面 - 儲存記憶
**時間碼**: 00:50 - 01:05  
**類型**: 螢幕錄製 + 動畫

**畫面內容**:
- 用戶輸入: "我喜歡喝冰拿鐵，少糖"
- AI 回應: "收到！我已經記住你的咖啡偏好了 ✓"
- 記憶儲存動畫（氣泡 → 膠囊 → Vault）

**技術需求**:
- [ ] 網頁介面錄製
- [ ] 準備動畫素材（記憶膠囊圖示）
- [ ] 添加發光效果

**動畫流程**:
```
1. 對話氣泡出現
2. 氣泡縮小成球體
3. 球體飛向 Vault 圖示
4. Vault 發光確認
5. 顯示 Solana tx 簽名（小字）
```
