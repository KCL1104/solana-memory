## Ethereum Ecosystem Developments

### ü§ñ ERC-8004: AI Agent Standard (MAJOR)

**Overview**: New standard for AI agents to find each other, prove identity, and build trust across systems

**Three Core Registries**:
1. **Identity Registry**: ERC-721 style on-chain identifier for each agent
   - Portable, transferable, censorship-resistant
   - Points to registration file (capabilities, protocols, endpoints)
   
2. **Reputation Registry**: Structured feedback about agent performance
   - Raw signals on-chain
   - Complex scoring off-chain
   - Public and reusable across apps
   
3. **Validation Registry**: Independent verification of agent work
   - Staked services, ML proofs, TEE attestation
   - Results stored on-chain for transparency

**Impact**: Enables global AI agent marketplace without intermediaries
- Agents can interoperate across vendors, chains, jurisdictions
- Ethereum becomes neutral infrastructure for autonomous agents
