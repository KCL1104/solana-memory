## 📍 分段詳細腳本

### 第一段：開場 Hook (00:00 - 00:15)

**【畫面】** 快速剪輯：重複的對話截圖，每次都在問「你喝咖啡嗎？」

**【視覺效果】** 
- 畫面中央大數字跳動：1, 5, 12, 23, 36, **47**
- 每一次跳動都伴隨一個聊天記錄閃過
- 第47次時畫面定格，加入故障效果(Glitch Effect)

**【字幕/字卡】**
- 主標：「這是你第 47 次告訴 AI 你喜歡咖啡」
- 副標：「但它還是忘記了」

**【音效】** 機械重複音效，逐漸加速，最後變成刺耳的雜訊聲

---

### 第二段：問題呈現 (00:15 - 00:35)

**【畫面】** 左右對比畫面 (Split Screen)

**左側 - 沒有記憶的 AI：**
- 每次對話都是全新的
- 聊天記錄顯示 "Hello, I'm Claude. How can I help you today?"
- 用戶重複輸入相同的背景資訊

**右側 - 有記憶的 AI (想像)：**
- AI 主動說：「早安！根據你之前的偏好...」

**【動畫過渡】** 中央出現一個問號動畫，然後變成 AgentMemory Logo

**【字幕/字卡】**
- 主標：「為什麼 AI 不能像人類一樣記住？」
- 副標：「每次重啟 = 失憶」

---

### 第三段：解決方案展示 (00:35 - 01:35)

#### 3.1 創建 Vault (00:35 - 00:50)

**【畫面】** 乾淨的 Terminal / Web UI 介面

**【操作展示】**
```bash
# 創建你的記憶庫
agent-memory vault create --name "my-preferences"
```

**【視覺效果】** 
- 指令輸入時有打字機效果
- 成功創建後顯示 Vault ID 和 Solana 地址
- 畫面右側彈出「✓ Vault Created」通知

**【字幕】** 「第一步：創建你的永久記憶庫」

---

#### 3.2 儲存第一筆記憶 (00:50 - 01:05)

**【畫面】** 用戶與 AI 對話介面

**【對話展示】**
```
User: 我喜歡喝冰拿鐵，少糖
AI: 收到！我已經記住你的咖啡偏好了 ✓

[畫面切換到記憶儲存的動畫]
```

**【視覺效果】**
- 對話氣泡縮小進入一個「記憶膠囊」
- 膠囊飛入 Vault 圖示
- 顯示 Solana 交易確認（快速閃過）

**【字幕】** 「儲存偏好，永久保存」

---

#### 3.3 儲存更多記憶 (01:05 - 01:20)

**【畫面】** 快速展示多種記憶類型

**【蒙太奇剪輯】**
| 時間 | 記憶類型 | 畫面 |
|------|----------|------|
| 01:05 | 工作習慣 | 「我習慣早上 9 點開始寫程式」 |
| 01:10 | 專案背景 | 「這個專案使用 React + Solana」 |
| 01:15 | 個人喜好 | 「我討厭蜘蛛」 |

**【視覺效果】** 
- 多個記憶膠囊飛入 Vault
- Vault 逐漸發光，顯示「記憶力 +3」

**【字幕】** 「習慣、專案、偏好... 全部記住」

---

#### 3.4 版本控制展示 (01:20 - 01:30)

**【畫面】** 展示記憶的歷史版本

**【操作展示】**
```
[記憶條目: 咖啡偏好]
├─ 版本 1: 我喜歡拿鐵
├─ 版本 2: 我喜歡冰拿鐵 ← 當前
└─ 版本 3: 我現在改喝美式 (草稿)
```

**【視覺效果】** Git 風格的版本樹圖形

**【字幕】** 「記憶也會進化，隨時回溯」

---

#### 3.5 跨 Agent 分享 (01:30 - 01:35)

**【畫面】** 兩個不同的 AI Agent 介面

**【動畫】** 記憶膠囊從一個 Agent 飛到另一個 Agent

**【字幕】** 「一次記憶，全平台同步」

### 第四段：魔法時刻 (01:35 - 02:05)

**【場景轉換】** 黑場 → 第二天早上

**【畫面】** 用戶打開電腦，啟動 AI

**【對話展示】**
```
[螢幕漸亮，顯示 AI 介面]

AI: 早安！☀️
AI: 要來杯冰拿鐵嗎？少糖那種

[用戶驚訝表情 - 可用 emoji 表示]

User: 你怎麼知道？！
AI: 我當然記得，這是昨天存的 😉
```

**【視覺高潮】**
- AI 回覆時，畫面底部顯示 Solana Transaction Signature
- 格式：「✓ On-chain verified: 5xJd...9mK2」
- 快速閃過區塊鏈瀏覽器畫面，證明數據真實上鏈

**【音效】** 溫暖的提示音 + 輕快的背景音樂進入

**【字幕】** 「第二天，它還記得」
